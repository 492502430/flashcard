1. AI 生成质量闭环

为每张卡保存来源段落、生成理由和质量评分；后台聚合“答案过长、问题泛、重复疑似”等问题，形成可优化队列。

2. 模板库运营

后台支持模板牌组上架、分类、封面、推荐位和导入转化统计，优先沉淀高频学习场景。

3. 用户增长与邀请

跟踪邀请人数、被邀请用户留存和生成牌组数，判断分享是否真的带来学习行为。

4. 风控与成本

监控 token 消耗、生成失败率、单用户异常导入量，后续可接入额度、会员和内容审核。
